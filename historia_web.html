História da Internet e da Web

  1. surgimento das redes de computadores

    O surgimento das redes de computadores ocorreu na década de 1960. Impulsionado por contextos de pesquisa e militares durante a Guerra Fria, o marco inicial foi a criação da ARPANET pela agência americana DARPA

  2. ARPANET

    A ARPANET (Advanced Research Projects Agency Network) foi a primeira rede de computadores do mundo, criada em 1969 pelo Departamento de Defesa dos Estados Unidos. Ela serviu como base tecnológica para a Internet moderna, introduzindo conceitos revolucionários como a comunicação por comutação de pacotes e o desenvolvimento do protocolo TCP/IP

  3. TCP/IP

    foi criado na década de 1970 pelo Departamento de Defesa dos Estados Unidos O sistema foi desenvolvido inicialmente para a ARPANET, uma rede experimental criada pelo Advanced Research Projects Agency (ARPA). A necessidade surgiu porque os militares queriam uma rede de comunicação robusta e descentralizada

  4. DNS

    O DNS (Sistema de Nomes de Domínio) surgiu em 1983 para resolver o problema de escalabilidade da internet. Criado por Paul Mockapetris, ele substituiu o antigo arquivo HOSTS.TXT, que exigia que todos os computadores fizessem o download manual de uma tabela centralizada com nomes e números

  5. criação da World Wide Web

    A World Wide Web (WWW) foi criada em 1989 pelo cientista da computação britânico Tim Berners-Lee, no laboratório do CERN (Organização Europeia para a Pesquisa Nuclear), na Suíça. O objetivo inicial era facilitar o compartilhamento descentralizado de informações entre cientistas e pesquisadores

  6. Tim Berners-Lee

    Sir Tim Berners-Lee inventou a World Wide Web enquanto trabalhava no CERN , o Laboratório Europeu de Física de Partículas, em 1989. Ele escreveu o primeiro cliente e servidor web em 1990. Suas especificações de URIs, HTTP e HTML foram aprimoradas à medida que a tecnologia web se disseminou.

  7. HTML

    O HTML surgiu em 1990, criado por Tim Berners-Lee ainda enquanto estava no CERN. O objetivo inicial era criar uma forma simples de organizar e compartilhar documentos de pesquisas científicas entre instituições, interligando textos por meio de hiperlinks

  8. HTTP

    O HTTP (Hypertext Transfer Protocol) surgiu em 1989, criado também por Tim Berners-Lee. Ele foi desenvolvido na CERN como um dos pilares fundamentais da World Wide Web para padronizar a comunicação entre navegadores (clientes) e servidores

  9. navegador

    O primeiro navegador web, chamado WorldWideWeb (depois renomeado para Nexus), foi criado em 1990 por Tim Berners-Lee novamente. Inicialmente, era uma ferramenta acadêmica simples que rodava apenas nos computadores da empresa NeXT

  10. Servidores web

    Os servidores web também surgiram em 1990, criados por Tim Berners-Lee. O marco ocorreu no CERN (Organização Europeia para a Pesquisa Nuclear), na Suíça, onde ele desenvolveu o primeiro software de servidor (httpd) e o primeiro navegador (WorldWideWeb)

  11. diferença entre Web 1.0 e aplicações Web modernas

    1. WEB 1.0

      A Web 1.0 focava somente em buscar e ler informações.

    2. WEB 2.0

      Já a Web 2.0 focou em ler, escrever, criar e interagir com o usuário final. Ficando famosa por ser chamada de web social participativa.

    3. WEB 3.0

      E para finalizar a Web 3.0 é a terceira geração da World Wide Web e representa uma visão de uma web descentralizada, atualmente em desenvolvimento. Se concentrando em ler, escrever e possuir.

  12. Citações Tim Berners Lee

    O objetivo da Web é servir à humanidade. Nós a construímos agora para que aqueles que venham a ela mais tarde possam criar coisas que nós mesmos não conseguimos imaginar.

    Eu só precisei pegar a ideia de hipertexto e conectá-la às ideias de TCP e DNS e — voilà! — surgiu a World Wide Web .
