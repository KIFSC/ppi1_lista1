✅ Atividade 2 – História da Internet e da Web em HTML
	Arquivo: historia_web.html
Crie uma página explicando a evolução da Internet e da Web.
A página deve apresentar uma linha do tempo textual contendo, no mínimo:
surgimento das redes de computadores;
ARPANET;
TCP/IP;
DNS;
criação da World Wide Web;
Tim Berners-Lee;
HTML;
HTTP;
navegador;
servidor web;
diferença entre Web 1.0 e aplicações Web modernas.
A página deve conter:
um menu de navegação para voltar ao index.html e acessar as demais páginas;
pelo menos duas imagens com <img>;
texto alternativo adequado nas imagens usando alt;
pelo menos três links externos para fontes de pesquisa;
uma citação curta ou referência a Tim Berners-Lee ou ao CERN;
uma lista de definição usando:
uma tabela com pelo menos 6 marcos históricos.
A tabela deve conter:
ano;
acontecimento;
importância para a Web ou Internet.
Use obrigatoriamente na página: 
h1, h2, h3, p, img, a, div, span, hr, em, strong, mark, sub, sup, ul, ol, li, dl, dt, dd, table, caption, thead, tbody, tfoot, tr, th, td 

✅ Atividade 3 – HTTP na prática: requisição e resposta
Arquivo:  http.html
Crie uma página explicando o que é o protocolo HTTP e como ele é usado na Web.
A página deve conter:
explicação sobre o papel do HTTP na comunicação entre navegador e servidor;
explicação sobre cliente, servidor, requisição e resposta;
descrição dos métodos GET e POST;
exemplo de uma requisição HTTP;
exemplo de uma resposta HTTP;
explicação de pelo menos 5 códigos de status HTTP, por exemplo:
200;
301 ou 302;
400;
403;
404;
500
